# Jazz Wordle

Simple Wordle clone themed around jazz musicians, albums and standards. 5-letter words only.

Run:

Open `index.html` in a browser (double-click or use a local dev server).

Notes:
- Use the `word length` selector to choose a 5–7 letter game.
- `js/words.js` contains the answer bank (only full words) and an expanded allowed list.
- Keyboard and physical keys supported.
